# v0.1.0 - Initial Test Release

First test release to validate the core application setup.

## Current features
- Vue 3 and Tailwind CSS based UI
- Cross-platform desktop support using Tauri
- HLS video playback
- Auto-update support
- Local SQLite database

## Development status
- Base architecture and tooling in place
- CI/CD pipelines configured for builds and releases

## Notes
This is an *experimental test build*. APIs, features, and behavior may change at any time.
Bug reports and feedback are welcome via the GitHub repository.
