# v1.1.0

This release introduces new video sources, custom player controls, and several quality-of-life improvements.

## New features

- Add support for Filemoon and Doodstream video sources
- Implement custom video player controls
- Add ability to mark a complete season as watched
- Add settings for the Updater

## Fixed

- Fix PlayerView scrolling issues on smaller screens
- Fix issue where watch progression was not consistently loaded on stream start
- Fix various UI theming issues
- Fix source name and missing genre metadata
- Fix software is crashing when starting without an internet connection
- Performing security audits of project dependencies and updating them
